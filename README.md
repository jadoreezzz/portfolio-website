# Portfolio Website

Интерактивный одностраничный сайт-портфолио — canvas с нодами проектов, зум/пан, всплывающие окна с вертикальными видео (9:16).

## Стек
Чистый статический сайт: один `index.html` (HTML + CSS + ванильный JS, без зависимостей и сборки).

## Структура
```
index.html                          # весь сайт
node content : ai and social media/  # видео-контент проектов (.mp4)
  dibrain studio project/
  huggito project/
```

## Локальный запуск
Открыть `index.html` в браузере, либо поднять любой статический сервер:
```bash
python3 -m http.server 8000
# затем http://localhost:8000
```

## Деплой
Статический сайт — деплоится как есть, без шага сборки. Корень публикации — папка проекта, входная точка `index.html`.

> Видео-файлы в `node content : ai and social media/` нужно положить реальными — в репозитории они могут быть плейсхолдерами.
